一个用于腾讯云服务的TypeScript SDK，提供了简洁易用的API接口。

## 功能特性

- 基于TypeScript开发，提供完整的类型定义
- 支持腾讯云API v3鉴权
- 使用axios作为HTTP客户端
- 支持ES模块导入
- 易于扩展的架构设计
